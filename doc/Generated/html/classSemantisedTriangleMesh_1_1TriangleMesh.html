std::vector<std::shared_ptr<Vertex> > SemantisedTriangleMesh::TriangleMesh::computeShortestPath ( std::shared_ptr< Vertex >  v1,
std::shared_ptr< Vertex >  v2,
const DistanceType  metric,
const bool  directed,
const bool  avoidUsed 
)

computeShortestPath approximate version of the famous Dijkstra algorithm for the shortest path computation. The method stops the search for better paths whenever it finds the target vertex v2

Parameters
v1the origin of the path
v2the target position of the path
metricthe metric to be used for the distance computation
avoidUsedif checked, gives ∞ weight to the already used arcs, to avoid intersections
directedif checked allows to reduce computation time choosing vertices in the frontier that are closer to the target in an Euclidean way (doesn't work well with very geodesically distant target vertices in high curvature meshes)
Returns
the shortest path (as list of successive vertices) connecting v1 and v2
